# Project Description:
This is a C++ STRUCTURED PROGRAMMING project that manages loan records.
The system allows adding, searching, updating, and deleting loan information
using file handling. It stores backup in binary file and current data in text file. This is a halal loan system without interests. This is a secured loan system specifically for the admin for record keeping.

# Features
- Set password for Security
- Add new loan record
- Update loan details
- Search loan record by ID
- Delete loan record by ID
- Search loan record by CNIC
- Delete loan record by CNIC
- ChangePassword
- Stores data in a text file
- Input validation
- Gives overview of the system's current status
- Pay installments
- Display all loans
- Add Balance to the current Balance
- Record History
- Display History
- Store Backup
- Display Backup
- Restore Backup Content/Data
- Login/Logout (Secured System By Password Verification)
